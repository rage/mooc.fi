### 1.Installing Java <a name="javan-asentaminen"></a>

1. Download the open development environment for Java 11, AdoptOpenJDK11, from [https://adoptopenjdk.net](https://adoptium.net/temurin/releases/?os=mac&version=11&package=jdk).

2. The page should be filtered to only show installation packages for macOS, for package type JDK and version 11 - LTS. Select the correct installer for your operating system's architecture type , and download the installation file with the .pkg-button.

3. When the download is complete, open the installation file.

4. Follow the installation instructions to install Java.

If in phase 3 you get a notification saying **"OpenJDK can't be opened because it is from an unidentified developer."**,
navigate to the directory you downloaded the installation file to in Finder, right-click the filename and select "Open" (see images below).

![macOS notification saying OpenJDK could not be opened because it is from an unidentified developer](macos-openjdk-not-trusted.webp)

![Finder view, where OpenJDK installer has been right clicked, and option Open is highlighed on the opened dropdown menu.](macos-openjdk-rightclick-open.webp)



###  2. Installing TMCBeans <a name="tmcbeansin-asentaminen"></a>

For this step, you must have macOS 10.13 or newer. If you have an older version of macOS, download the zip version of TMCBeans by following the instructions in the ZIP tab.

1. Download file [https://download.mooc.fi/tmcbeans/installers/tmcbeans_1.4.0_installer.dmg](https://download.mooc.fi/tmcbeans/installers/tmcbeans_1.4.0_installer.dmg).
2. When the download is complete, open the file. The following window should open:

![View from tmcbeans.dmg installer open in Finder.](macos-tmcbeans-installer.webp)

If the window does not open, click the tmcbeans-drive (see the image below) on Finder's sidebar.

![Finder view showing tmcbeans-drive mounted on the left sidebar.](macos-tmcbeans-finder-sidebar-dmg.webp)

3. In the view that opens, drag the tmcbeans.app file to the Applications folder.
4. You can now close the dmg-installation file. Click the eject button at the left side of Finder (see the image below).

![Finder view. Eject button next to the tmcbeans-drive on the left sidebar of Finder has been circled.](macos-tmcbeans-finder-sidebar-eject-dmg.webp)

If you get a notification saying **"tmcbeans.app can't be opened because it is from an unidentified developer"** when you try to open TMCBeans, navigate to the Applications folder in Finder,  and try opening TMCBeans by right clicking it and selecting Open (see the images below).

![macOS notification saying tmcbeans.app file cannot be opened because it is from an unidentified developer.](macos-tmcbeans-not-trusted.webp)

![Finder view of the Applications directory. Tmcbeans.app file has been right clicked, and Open opetion is highlighted on the dropdown menu.](macos-tmcbeans-rightclick-open.webp)
