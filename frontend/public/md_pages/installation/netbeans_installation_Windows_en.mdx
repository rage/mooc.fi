### Installing Java <a name="javan-asentaminen"></a>

Download AdoptOpenJDK11, open development environment for Java 11, from [https://adoptopenjdk.net](https://adoptium.net/temurin/releases/?os=windows&version=11&package=jdk).

The page should be filtered to only show installation packages for Windows, for package type JDK and version 11 - LTS. Select the correct installer for your operating system's architecture type (x64 for 64-bit operating system, x86 for 32-bit operating system), and download the installation file with the .msi-button.

**NB!** When installing Java with the installation file you just downloaded, in the Custom Setup phase set both "Set JAVA_HOME variable" and "JavaSoft (Oracle) registry keys" to "Entire feature will be installed on local hard drive" (see the images below).


![Custom Setup phase of AdoptOpenJDK installer, where Set JAVA_HOME variable is set to Entire feature will be installed on local hard drive.](adoptopenjdk-set-java-home.webp)

![Custom Setup phase of AdoptOpenJDK installer, where JavaSoft (Oracle) registry keys is set to Entire feature will be installed on local hard drive. ](adoptopenjdk-set-registry-keys.webp)


### Installing TMCBeans <a name="tmcbeansin-asentaminen"></a>

1. Download file [https://download.mooc.fi/tmcbeans/installers/tmcbeans_1.5.0_installer.exe](https://download.mooc.fi/tmcbeans/installers/tmcbeans_1.5.0_installer.exe).
2. When the download is complete, double click the file to open the installer.
3. Follow the instructions on your screen.
