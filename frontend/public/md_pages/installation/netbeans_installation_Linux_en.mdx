### Installing Java <a name="javan-asentaminen"></a>

#### 1. Install OpenJDK-package.

You can install OpenJDK with the following terminal command (works for Ubuntu):

> `sudo apt-get install openjdk-11-jdk`

#### 2. Switch the active version of Java

Use the command:

> `sudo update-alternatives --config java`

Type the number of the option, which says Java 11 (for example on this computer "/usr/lib/jvm/java-11-openjdk-amd64/bin/java"), and press enter.

If there are multiple options with Java 11, both with status "auto mode" and "manual mode", select "manual mode".

</SectionBox>

<SectionBox>

###  Installing TMCBeans <a name="tmcbeansin-asentaminen"></a>

In terminal, run command:

> `sudo snap install --classic tmcbeans`

Type your password to the terminal when prompted and press enter.

You can now start TMCBeans either from the terminal with command `tmcbeans` or by starting it from Activities.

**If snap install does not work on your computer**, follow these instructions to install snap: [Installing snapd](https://snapcraft.io/docs/installing-snapd). If your operating system does not support snap, download the zip version by following the instructions on the ZIP tab.

**If the command tmcbeans does not work on your computer**, try to launch the program with command `snap run tmcbeans`.
