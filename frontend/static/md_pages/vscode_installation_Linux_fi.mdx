Seuraa alla olevia ohjeita asentaaksesi Visual Studio Code (VSCode) ja TestMyCode (TMC) -laajennus VSCodeen. TestMyCode -laajennus tarvitsee Javan suoritysympäristöä toimiakseen oikein.

Mikäli sinulla on käytössäsi Tietojenkäsittelytieteen osaston laite, voit siirtyä suoraan [TestMyCode -laajennuksen asentamiseen](#TestMyCode-asentaminen).

###  Visual Studio Coden asentaminen <a name="VSCoden-asentaminen"></a>

1. Lataa asennusohjelma osoitteesta [https://code.visualstudio.com/download](https://code.visualstudio.com/download)
2. Kun tiedoston lataaminen on valmis, tuplaklikkaa tiedostoa avataksesi asennusohjelma.
3. Tämän jälkeen seuraa ruudulle tulevia ohjeita.

### Javan Asentaminen <a name="javan-asentaminen"></a>

**HUOM!** TestMyCode laajennus lataa käyttöönsä Javan suoritusympäristön, mikäli tietokoneellasi ei ole Javaa asennettuna.  
On kuitenkin suositeltavaa asentaa Java itse, ennen kuin jatkat TestMyCode-laajennuksen asentamista. 
Mikäli koneellasi on jo Javan suoritusympäristö asennettuna, voit siirtyä kohtaan [TestMyCode -laajennuksen asentamiseen](#TestMyCode-asentaminen).  

#### 1. Asenna OpenJDK-paketti.

Ubuntussa sen voi tehdä komentoriviltä seuraavalla komennolla:

> `sudo apt-get install openjdk-11-jdk`

#### 2. Vaihda Javan aktiivinen versio

Käytä komentoa:

> `sudo update-alternatives --config java`

Kirjoita komentoriville sen vaihtoehdon numero, jossa lukee Java 11 (esim kirjoittajan koneella "/usr/lib/jvm/java-11-openjdk-amd64/bin/java"), ja paina enter.

Jos Java 11 on listalla useampaan otteeseen, sekä statuksella "auto mode" että "manual mode", valitse mieluummin "manual mode".

### Pythonin asentaminen <a name="python-asentaminen"></a>

Python3 ja Python3 pip asentaminen voi tehdä komentoriviltä seuraavalla komennolla (Ubuntu):

> `sudo apt-get install python3 python3-pip`

### TestMyCode -laajennuksen asentaminen VSCodeen <a name="TestMyCode-asentaminen"></a>

1. Avaa Visual Studio Code ohjelma.  
2. Ohjelman vasemmassa sivupalkissa näkyy ikoneita, valitse sieltä Extensions -näkymä (kuvassa kohta 1).  
3. Kirjoita hakukenttään **TestMyCode** (kuvassa kohta 2).  
4. Valitse **TestMyCode** hakutuloksesta.
5. Paina "Install" -painiketta (kuvassa kohta 3).

Ensimmäisen käynnistyksen yhteydessä laajennus lataa sen tarvitsemat komponentit toimiakseen (tässä menee hetki).

## Ohjelmoinnin aloittaminen <a name="ohjelmoinnin-aloittaminen"></a>
Kun olet asentanut VSCoden, Javan suoritusympäristön ja TestMyCode -lisäosan, seuraavina askeleina on **Mooc.fi-tunnuksen** luominen sekä ohjelmoinnin aloittaminen. 
Mooc.fi-tunnus luodaan, jotta voimme pitää kirjaa tekemistäsi tehtävistä.

### MOOC.fi -tunnuksen luominen <a name="mooc.fi-tunnuksen-luominen"></a>
Luo itsellesi MOOC.fi tunnus kurssimateriaalissa.

### Kirjautuminen ohjelmointiympäristöön <a name="kirjautuminen-ohjelmointiympäristöön"></a>

Käynnistä TestMyCode-laajennus painamalla TMC-kuvaketta vasemmassa palkissa (kuvassa kohta 1).

Kirjaudu sisään painamalla sivuun ilmestynyttä **Log in** painiketta (kuvassa kohta 2).

Käytä edellisessä kohdassa luomaasi MOOC.fi tunnusta kirjautumiseen.

<img src="/static/images/installation/vscode/TMC_login.png"
alt="Ohjelmointiympäristöön kirjautuminen"
width="100%" height="100%"/>

### Organisaation valinta <a name="organisaation-valinta"></a>
Kirjauduttuasi sisään esiin tulee näkymä, joka listaa valitsemasi kurssit (aluksi se on tyhjä). 
Mikäli My courses -näkymä ei ole aktiivinen, pääset sinne painamalla "My courses" (kuvassa kohta 1).  

Kursseja voit lisätä painamlla **Add new course** painiketta (kuvassa kohta 2).

Painamalla **Add new course** painiketta laajennus näyttää kaikki mahdolliset organisaatiot, jotka tarjoavat kursseja.
Valitse organisaatioksi kurssimateriaalissa mainittu organisaatio (kuvassa kohta 3).

Esimerkiksi kurssilla "Ohjelmoinnin MOOC 2020" organisaationa on "MOOC".

<img src="/static/images/installation/vscode/select-org.png"
alt="Organisaation valinta"
width="100%" height="100%"/>

### Kurssin valinta <a name="kurssin-valinta"></a>

Valitse tämän jälkeen haluamasi kurssi. Kurssin nimi on mainittu kurssimateriaalissa.

### Ensimmäisen tehtävän tekeminen <a name="ensimmäisen-tehtävän-tekeminen"></a>

Käydään läpi esimerkki, miten saat tehtyä kurssin "Ohjelmoinnin MOOC 2020" toisen tehtävän.
Vaikka olisit tehnyt tehtäviä jollakin muulla kurssilla, lue ohjeet silti loppuun sillä ne sisältävät tärkeää tietoa ohjelmointiympäristön toiminnasta.

#### Tehtävien lataaminen

Kun olet kirjautunut ohjelmointiympäristöön sekä valinnut kurssin, klikkaa "My courses" -näkymään ilmestynyttä kurssia `mooc-2020-ohjelmointi`.

Paina sivulla näkyvää __Download exercises__ -painiketta.  
Lataa haluamasi tehtävät -- tehtävät liittyvät valitsemasi kurssin kurssimateriaaliin.  
Kun lataus on valmis, valitse __Return to My courses__.  

Nyt tehtävät ovat ladattuna ohjelmointiympäristössä.  
Paina __Open exercise workspace__.
<img src="/static/images/installation/vscode/open_exercise_workspace.png"
alt="Workspacen avaaminen"
width="100%" height="100%"/>

#### Ohjelmointitehtävän tekeminen

Siirry tehtävänäkymään painamalla Explorer -painiketta vasemmassa yläkulmassa (kuvassa kohta 1).  
Painamalla "Exercises" (kuvassa kohta 2) näet kaikki valitsemasi organisaatiot ja kurssit, sekä niiden tehtävät.  
Etsi sieltä tehtäväkansio "Osa01_02.AdaLovelace", organisaation "MOOC" ja kurssin "mooc-2020-ohjelmointi" alta.  
<img src="/static/images/installation/vscode/click_exercises.png"
alt="Tehtävän avaaminen"
width="100%" height="100%"/>

**HUOM!**
Ensimmäisen kerran kun avaat Visual Studio Codessa ohjelmointitehtävän saatat saada suosituksia asennettavista lisäosista, nämä on asennettava valitsemalla __Install__, jotta lähdekoodia pystyy ajamaan.
<img src="/static/images/installation/vscode/python_recommendation.png"
alt="Python lisäosa"
width="100%" height="100%"/>

Etsi tehtävän sisältä lähdekooditiedosto "AdaLovelace.java".
Lähdekooditiedostot löytyvät tehtäväpohjan kansiosta __"src"__.

Saat tehtävän auki editorinäkymään klikkaamalla tiedostoa AdaLovelace.java.  
Editorinäkymä on keskellä näkyvä tekstieditori, missä voit tehdä muutoksia ohjelman lähdekoodiin.


Muokkaa lähdekoodia siten, että editorissa on seuraava teksti:

```java
    public class AdaLovelace {
        public static void main(String[]args){
            System.out.println("Ada Lovelace :D");
        }
    }
```

### Lähdekoodin ajaminen <a name="lähdekoodin-ajaminen"></a>

Mikäli asensit tarvittavat ohjelmat ja lisäosat Visual Studio Codeen, voit suorittaa lähdekoodia painamalla VSCodessa oikealla ylänurkassa olevaa vihreää start painiketta.
Tämä aukaisee VSCodessa terminaalin lähdekoodieditorin alle.

<img src="/static/images/installation/vscode/python_run_terminal.png"
alt="Python aja terminaalissa"
width="100%" height="100%"/>

Jos muokkasit AdaLovelace.java-tiedostoa edellä kuvatun näköiseksi, ohjelman tulostus on nyt seuraavanlainen.

> `Ada Lovelace :D`

### Lähdekoodin testaaminen <a name="lähdekoodin-testaaminen"></a>

Tehtävissä tulee usein mukana automaattisia testejä, jotka antavat palautetta tehtävän onnistumisesta.

Testit ajetaan klikkaamalla oikealla yläkulmassa näkyvää "TMC - Run tests" painiketta.

<img src="/static/images/installation/vscode/tmc_run_test.png"
alt="TMC Testien ajaminen"
width="100%" height="100%"/>

Kun suoritat yllä luodun AdaLovelace-tehtävän testit, testien pitäisi antaa virheilmoitus.
Virheilmoitusta tarkastelemalla saamme selville, että tulostus ei ollut toivottu.
Tulostuksen olisi pitänyt olla muotoa "Ada Lovelace".

Ohjelman tulostus on muotoa "Ada Lovelace" kun muokkaat lähdekoodin seuraavanlaiseksi.

```java
    public class AdaLovelace {

        public static void main(String[] args) {
            System.out.println("Ada Lovelace");
        }
    }
```

Aja nyt tehtävään liittyvät testit uudestaan. Tällä kertaa testit menevät läpi ja TMC kysyy lähetetäänkö tehtävä palvelimelle.

### Tehtävän lähettäminen palvelimelle <a name="tehtävän-lähettäminen-palvelimelle"></a>

Tehtävän lähettäminen palvelimelle tarkoittaa tehtävän palauttamista arvostelua varten.
Tehtävät tarkastetaan lähes poikkeuksetta automaattisesti, joten saat tehtävistä pisteet (tai mahdollisen virheviestin) nopeasti.

Tehtävän voi palauttaa joko testien ajamisen jälkeen kun testit menevät läpi, tai erikseen TMC-menusta (sijaitsee oikeassa ylänurkassa) 
olevaa nappia painamalla ja valitsemalla __Submit to server__.

<img src="/static/images/installation/vscode/tmc_menu.png"
alt="TMC Menu"
width="100%" height="100%"/>

Kun tehtävä on lähetetty palvelimelle ja siihen liittyvät testit menevät läpi, saat tehtävästä pisteet sekä mahdollisen lyhyen kyselyn. Vastaathan aina kyselyihin, sillä vastaukset auttavat kurssien kehityksessä.

### Pistetilanteen tarkistaminen <a name="pistetilanteen-tarkistaminen"></a>

Voit tarkastaa oman pistetilanteesi kirjautumalla osoitteessa [tmc.mooc.fi](https://tmc.mooc.fi) olevaan verkkopalveluun. Verkkopalveluun kirjaudutaan Mooc.fi-tunnuksilla.
