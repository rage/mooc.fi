Seuraa alla olevia ohjeita asentaaksesi Visual Studio Code (VSCode) ja TestMyCode (TMC) -laajennus VSCodeen. TestMyCode -laajennus tarvitsee Javan suoritysympäristöä toimiakseen oikein.

Mikäli sinulla on käytössäsi Tietojenkäsittelytieteen osaston laite, voit siirtyä suoraan [TestMyCode -laajennuksen asentamiseen](#TestMyCode-asentaminen).

###  Visual Studio Coden asentaminen <a name="VSCoden-asentaminen"></a>

1. Lataa asennusohjelma osoitteesta [https://code.visualstudio.com/download](https://code.visualstudio.com/download)
2. Kun tiedoston lataaminen on valmis, tuplaklikkaa tiedostoa avataksesi asennusohjelma.
3. Tämän jälkeen seuraa ruudulle tulevia ohjeita.

### Javan Asentaminen <a name="javan-asentaminen"></a>

**HUOM!** TestMyCode laajennus lataa käyttöönsä Javan suoritusympäristön, mikäli tietokoneellasi ei ole Javaa asennettuna.  
On kuitenkin suositeltavaa asentaa Java itse, ennen kuin jatkat TestMyCode-laajennuksen asentamista. 
Mikäli koneellasi on jo Javan suoritusympäristö asennettuna, voit siirtyä kohtaan [TestMyCode -laajennuksen asentamiseen](#TestMyCode-asentaminen).  

#### 1. Asenna OpenJDK-paketti.

Ubuntussa sen voi tehdä komentoriviltä seuraavalla komennolla:

> `sudo apt-get install openjdk-11-jdk`

#### 2. Vaihda Javan aktiivinen versio

Käytä komentoa:

> `sudo update-alternatives --config java`

Kirjoita komentoriville sen vaihtoehdon numero, jossa lukee Java 11 (esim kirjoittajan koneella "/usr/lib/jvm/java-11-openjdk-amd64/bin/java"), ja paina enter.

Jos Java 11 on listalla useampaan otteeseen, sekä statuksella "auto mode" että "manual mode", valitse mieluummin "manual mode".

### Pythonin asentaminen <a name="python-asentaminen"></a>

Python3 asentaminen voi tehdä komentoriviltä seuraavalla komennolla (Ubuntu):

> `sudo apt-get install python3`

### TestMyCode -laajennuksen asentaminen VSCodeen <a name="TestMyCode-asentaminen"></a>

1. Avaa Visual Studio Code ohjelma.  
2. Ohjelman vasemmassa sivupalkissa näkyy ikoneita, valitse sieltä Extensions -näkymä (kuvassa kohta 1).  
3. Kirjoita hakukenttään **TestMyCode** (kuvassa kohta 2).  
4. Paina "Install" -painiketta (kuvassa kohta 3).

Ensimmäisen käynnistyksen yhteydessä laajennus lataa sen tarvitsemat komponentit toimiakseen (tässä menee hetki).
