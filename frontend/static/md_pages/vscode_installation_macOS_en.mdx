Follow the instructions below to download Visual Studio Code (VSCode) and TestMyCode (TMC) extension for VSCode. 

You need to have administrator priviledges on your computer to install the necessary programs.  
If you are using a computer provided by the computer science department you can skip directly to [Installing TestMyCode extension](#installing-tmc) section.

###  Installing Visual Studio Code <a name="installing-vscode"></a>

1. Download the Mac installation file from [https://code.visualstudio.com/download](https://code.visualstudio.com/download).
2. Once download is completed, double-click the zip file to open it.
3. You should now see the Visual Studio Code application, you should move this application to the __Applications__ folder on you computer.

### Installing TestMyCode Extension for VS Code <a name="installing-tmc"></a>

1. Open Visual Studio Code.  
2. At the left you can see icons, choose the Extension view (see picture point 1).  
3. Type **TestMyCode** in the search field (see picture point 2).  
4. Press the search result **TestMyCode**.
5. Press "Install" (see picture point 3).  

The first initilization of the extension will download some necessary components (this will take a while).

### Installing Python environment <a name="installing-python"></a>

For courses using Python you will need Python and Python Extension for VS Code to run your source code.  
Please install Python following the instructions below.

#### Installing Python <a name="python3"></a>

1. Download Python3 from [https://www.python.org/downloads/mac-osx/](https://www.python.org/downloads/mac-osx/)
2. Choose from __Stable Release__ the latest (uppermost) "Download __macOS 64-bit installer__" link.
3. When the download is complete, open the installation file.
4. Follow the installation instructions to install Python.
5. Once the installation has completed successfully, you can close the window. (Certificates are not necessary)

#### Installing Python Extension for VS Code <a name="python-extension"></a>

1. Open Visual Studio Code
2. On the left bar, press "Extensions" tab (in pic 1.)
3. In the search field, type `Python` and select the extension by Microsoft (in pic 2.)
4. Press `Install` button (in pic 3.)

### Installing Java environment <a name="java-environment"></a>

For courses using Java you will need Java Development kit & Runtime Environment, Java Extension Pack for Visual Studio Code and Maven to build and run your source code.  
Please install Java following the instructions below.

#### Installing Java <a name="adopt-open-jdk"></a>

1. Download the open development and runtime environment for Java 11, AdoptOpenJDK11, from [https://adoptopenjdk.net](https://adoptopenjdk.net/?variant=openjdk11&jvmVariant=hotspot).
2. Select OpenJDK 11 (LTS) and HotSpot. Then click the "Latest release" button to download Java.
3. When the download is complete, open the installation file.
4. Follow the installation instructions to install Java.

If in phase 3 you get a notification saying **"OpenJDK can't be opened because it is from an unidentified developer."**,
navigate to the directory you downloaded the installation file to in Finder, right-click the filename and select "Open" (see images below).

#### Installing Maven and configuring settings <a name="maven-configuring"></a>

You can install Maven by following the instructions below.  

1. Download the latest Binary zip archive, `apache-maven-3.x.y-bin.zip`, from [https://maven.apache.org/download.cgi](https://maven.apache.org/download.cgi).
2. Extract the archive to your desired location, e.g. `/usr/local/`, this will create automatically a folder named `apache-maven-3.x.y`.
3. If you installed the Java Extension Pack in VS Code, it comes with `Maven for Java`. Set Maven path in this extension as below:
    * Open Visual Studio Code
    * Press at top left "File -> Preferences -> Settings" or use the hotkey: `CTRL + ,`
    * Find `maven.executable.path` (pic below 1.) and set Maven path (pic below 2.) as:  
    `<your extracted location/apache-maven-3.x.y/bin/mvn`  
    for example with version 3.6.3 of Maven extracted to `/usr/local/` the path would be:  
    `/usr/local/apache-maven-3.6.3/bin/mvn`
