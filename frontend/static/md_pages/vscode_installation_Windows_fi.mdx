Seuraa alla olevia ohjeita asentaaksesi Visual Studio Code (VSCode) ja TestMyCode (TMC) -laajennus VSCodeen. TestMyCode -laajennus tarvitsee Javan suoritysympäristöä toimiakseen oikein.

Tarvitset järjestelmävalvojaoikeudet tietokoneellesi asentaaksesi tarvittavat ohjelmat.  
Mikäli sinulla on käytössäsi Tietojenkäsittelytieteen osaston laite, voit siirtyä suoraan [TestMyCode -laajennuksen asentamiseen](#TestMyCode-asentaminen).

###  Visual Studio Coden asentaminen <a name="VSCoden-asentaminen"></a>

Kurssia varten tulisi asentamaan Visual Studio Code alla olevan linkin takaa. Visual Studio Code ei ole sama ohjelma kuin Microsoft Visual Studio.  

1. Lataa Windowsin asennusohjelma osoitteesta [https://code.visualstudio.com/download](https://code.visualstudio.com/download)
2. Kun tiedoston lataaminen on valmis, tuplaklikkaa tiedostoa avataksesi asennusohjelma.
3. Tämän jälkeen seuraa ruudulle tulevia ohjeita.

### Javan Asentaminen <a name="javan-asentaminen"></a>

TestMyCode -laajennus tarvitsee käyttöönsä Javan. Saat asennettua Javan käyttöösi seuraamalla ohjeita alla.

1. Lataa Java 11:n avoin kehitys- ja suoritusympäristö AdoptOpenJDK11 osoitteesta [https://adoptopenjdk.net](https://adoptopenjdk.net/?variant=openjdk11&jvmVariant=hotspot).
2. Valitse sivulta OpenJDK 11 (LTS) ja HotSpot. Sitten voit painaa Latest release -nappia ladataksesi Javan.

**HUOM!** Kun asennat Javaa lataamastasi asennustiedostosta, aseta Custom Setup -vaiheessa sekä "Set JAVA_HOME variable" (kuva 1, kohta 1) että "JavaSoft (Oracle) registry keys" -kohtiin (kuva 2, kohta 2) "Entire feature will be installed on local hard drive".  
Kun nämä on asetettu, paina "Next" (kuva 2, kohta 3).

### Pythonin asentaminen <a name="python-asentaminen"></a>

1. Mene osoitteeseen [https://www.python.org/downloads/](https://www.python.org/downloads/)
2. Lataa viimeisin saatavilla oleva versio Python 3:sta.
3. Käynnistä asennus tuplaklikkaamalla asennustiedostoa. 
4. Valitse asennusvaiheessa "Add Python 3.x to PATH" (kuvassa kohta 1).
5. Valitse "Install now" (kuvassa kohta 2).
6. Kun asennus on valmis, voit sulkea asennusikkunan.

### TestMyCode -laajennuksen asentaminen VSCodeen <a name="TestMyCode-asentaminen"></a>

1. Avaa Visual Studio Code ohjelma.  
2. Ohjelman vasemmassa sivupalkissa näkyy ikoneita, valitse sieltä Extensions -näkymä (kuvassa kohta 1).  
3. Kirjoita hakukenttään **TestMyCode** (kuvassa kohta 2).  
4. Valitse **TestMyCode** hakutuloksesta.
5. Paina "Install" -painiketta (kuvassa kohta 3).

Ensimmäisen käynnistyksen yhteydessä laajennus lataa sen tarvitsemat komponentit toimiakseen (tässä menee hetki).
