### Javan Asentaminen <a name="javan-asentaminen"></a>

Lataa Java 11:n avoin kehitysympäristö AdoptOpenJDK11 osoitteesta [https://adoptopenjdk.net](https://adoptopenjdk.net/?variant=openjdk11&jvmVariant=hotspot).

Valitse sivulta OpenJDK 11 (LTS) ja HotSpot. Sitten voit painaa Latest release -nappia ladataksesi Javan.

**HUOM!** Kun asennat Javaa lataamastasi asennustiedostosta, aseta Custom Setup -vaiheessa sekä "Set JAVA_HOME variable" että "JavaSoft (Oracle) registry keys" -kohtiin "Entire feature will be installed on local hard drive". (katso allaolevat kuvat)

###  TMCBeansin Asentaminen <a name="tmcbeansin-asentaminen"></a>

1. Lataa tiedosto [https://download.mooc.fi/tmcbeans/installers/tmcbeans_1.3.0_installer.exe](https://download.mooc.fi/tmcbeans/installers/tmcbeans_1.3.0_installer.exe).
2. Kun tiedoston lataaminen on valmis, tuplaklikkaa tiedostoa avataksesi asennusohjelman.
3. Tämän jälkeen seuraa ruudulle tulevia ohjeita.
