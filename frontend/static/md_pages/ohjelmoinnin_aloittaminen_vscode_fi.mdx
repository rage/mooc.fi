## Ohjelmoinnin aloittaminen <a name="ohjelmoinnin-aloittaminen"></a>
Kun olet asentanut VSCoden, Javan, Pythonin ja TestMyCode -lisäosan, seuraavina askeleina on **Mooc.fi-tunnuksen** luominen sekä ohjelmoinnin aloittaminen. 
Mooc.fi-tunnus luodaan, jotta voimme pitää kirjaa tekemistäsi tehtävistä.

### MOOC.fi -tunnuksen luominen <a name="mooc.fi-tunnuksen-luominen"></a>
Mikäli sinulla ei ole MOOC.fi tunnusta, luo itsellesi MOOC.fi tunnus kurssimateriaalissa tai tämän sivun oikeassa yläkulmassa olevasta __Luo uusi tunnus__ -painikkeesta.

### Kirjautuminen ohjelmointiympäristöön <a name="kirjautuminen-ohjelmointiympäristöön"></a>

Käynnistä TestMyCode-laajennus painamalla TMC-kuvaketta vasemmassa palkissa (kuvassa kohta 1).

Kirjaudu sisään painamalla sivuun ilmestynyttä **Log in** painiketta (kuvassa kohta 2).

Käytä edellisessä kohdassa luomaasi MOOC.fi tunnusta kirjautumiseen.

<img src="/static/images/installation/vscode/TMC_login.png"
alt="Ohjelmointiympäristöön kirjautuminen"
width="100%" height="100%"/>

### Organisaation valinta <a name="organisaation-valinta"></a>
Kirjauduttuasi sisään esiin tulee näkymä, joka listaa valitsemasi kurssit (aluksi se on tyhjä). 
Mikäli My courses -näkymä ei ole aktiivinen, pääset sinne painamalla "My courses" (kuvassa kohta 1).  

Kursseja voit lisätä painamalla **Add new course** painiketta (kuvassa kohta 2).

Painamalla **Add new course** painiketta laajennus näyttää kaikki mahdolliset organisaatiot, jotka tarjoavat kursseja.
Etsi kurssimateriaalissa mainittu organisaatio ja valitse tämä (kuvassa kohta 3).

Esimerkiksi kurssilla "Ohjelmoinnin perusteet Python, kesä 2020" organisaationa on "Helsingin Yliopisto".

<img src="/static/images/installation/vscode/select-org.png"
alt="Organisaation valinta"
width="100%" height="100%"/>

### Kurssin valinta <a name="kurssin-valinta"></a>

Valitse tämän jälkeen haluamasi kurssi. Kurssin nimi on mainittu kurssimateriaalissa. Listatut kurssit riippuvat organisaatiosta.

### Ensimmäisen tehtävän tekeminen <a name="ensimmäisen-tehtävän-tekeminen"></a>

Käydään läpi esimerkki, miten saat tehtyä kurssin "Ohjelmoinnin perusteet Python, kesä 2020" Hello Visual Studio Code -tehtävän.
Vaikka olisit tehnyt tehtäviä jollakin muulla kurssilla, lue ohjeet silti loppuun sillä ne sisältävät tärkeää tietoa ohjelmointiympäristön toiminnasta.
Tehtävänannot löytyvät kurssimateriaalista.

#### Tehtävien lataaminen

Kun olet kirjautunut ohjelmointiympäristöön sekä valinnut kurssin `Ohjelmoinnin perusteet Python, kesä 2020`, klikkaa kurssia "My courses" -näkymässä.

Saavut kurssinäkymään. Paina sivulla näkyvää __Download__ -painiketta osan 4 kohdalla (kuva alla), niin ohjelma aloittaa tämän osan tehtävien lataamisen tietokoneellesi.
Tehtävien latauksen etenemisen näet osan kohdalla olevasta "Downloaded" -lukemasta.
Odotathan, että tehtävät ovat latautuneet ennen kuin siirryt eteenpäin ohjeissa.

Kun tehtävät ovat latautuneet, paina __Open exercise workspace__, mikäli painike ei näy tehtäväympäristö on jo avattuna ja voit siirtyä eteenpäin.

#### Ohjelmointitehtävän tekeminen

Jos et ole tehtävänäkymässä, eli et näe "Exercises" vasemmalla kuten kuvassa alla kohdassa 2, niin pääset sinne painamalla Explorer -painiketta vasemmassa yläkulmassa (kuvassa kohta 1). 

Painamalla "Exercises" (kuvassa kohta 2) näet valitsemasi organisaation lyhenteen (hy) ja kurssinimen (hy-python-kesa-20), sekä lataamasi tehtävät.

**HUOM!**
Ensimmäisen kerran kun avaat Visual Studio Codessa ohjelmointitehtävän saatat saada suosituksia asennettavista lisäosista, nämä on asennettava valitsemalla __Install__, jotta lähdekoodia pystyy ajamaan.
<img src="/static/images/installation/vscode/python_recommendation.png"
alt="Python lisäosa"
width="100%" height="100%"/>

Etsi tehtävän sisältä lähdekooditiedosto "hello_visualstudio_code.py".
Lähdekooditiedostot löytyvät tehtäväpohjan kansiosta __"src"__.

Saat tehtävän auki editorinäkymään klikkaamalla tiedostoa hello_visualstudio_code.py.  
Editorinäkymä on keskellä näkyvä tekstieditori, missä voit tehdä muutoksia ohjelman lähdekoodiin.


Muokkaa lähdekoodia siten, että editorissa on seuraava teksti:

```python
editori = input("editori: ")
if editori == "word" or editori == "notepad":
    print("surkea")
```

### Lähdekoodin ajaminen <a name="lähdekoodin-ajaminen"></a>

Mikäli asensit tarvittavat ohjelmat ja lisäosat Visual Studio Codeen, voit suorittaa lähdekoodia painamalla VSCodessa oikealla ylänurkassa olevaa vihreää start painiketta.

<img src="/static/images/installation/vscode/python_run_terminal.png"
alt="Python aja terminaalissa"
width="100%" height="100%"/>

Tämä aukaisee VSCodessa terminaalin lähdekoodieditorin alle, mihin ohjelman tulostus ja syötteenanto tulevat.
<img src="/static/images/installation/vscode/terminal-vscode.png"
alt="Terminaali"
width="100%" height="100%"/>

Jos muokkasit hello_visualstudio_code.py-tiedostoa edellä kuvatun näköiseksi, ohjelman tulostus on nyt seuraavanlainen syötteellä `notepad`.

```
editori: notepad
surkea
```

### Lähdekoodin testaaminen <a name="lähdekoodin-testaaminen"></a>

Tehtävissä tulee usein mukana automaattisia testejä, jotka antavat palautetta tehtävän onnistumisesta.

Testit ajetaan klikkaamalla TMC:ssä __testinappia__, joka näyttää seuraavalta.

<img src="/static/images/installation/vscode/tmc_run_test.png"
alt="TMC Testien ajaminen"
width="100%" height="100%"/>

Kun suoritat yllä luodun Hello Visual Studio Code-tehtävän testit, testien pitäisi antaa virheilmoitus.
Virheilmoitusta tarkastelemalla saamme selville, että tulostus ei ollut oikein kaikilla syötteillä.

Ohjelman tulostus eri syötteillä on oikein, kun muokkaat lähdekoodin seuraavanlaiseksi.

```python
while True:
    editori = input("editori: ")
    if editori.lower() == "visual studio code":
        break
    if editori == "word" or editori == "notepad":
        print("surkea")
    else:
        print("ei ole hyvä")
print("loistava valinta!")
```

Aja nyt tehtävään liittyvät testit uudestaan. Tällä kertaa testit menevät läpi ja voit lähettää tehtävän TMC-palvelimelle valitsemalla __Submit to server__.

### Tehtävän lähettäminen palvelimelle <a name="tehtävän-lähettäminen-palvelimelle"></a>

Tehtävän lähettäminen palvelimelle tarkoittaa tehtävän palauttamista arvostelua varten.
Tehtävät tarkastetaan lähes poikkeuksetta automaattisesti, joten saat tehtävistä pisteet (tai mahdollisen virheviestin) nopeasti.

Tehtävän voi palauttaa joko testien ajamisen jälkeen kun testit menevät läpi, tai erikseen TMC-menusta (sijaitsee oikeassa ylänurkassa) 
olevaa nappia painamalla ja valitsemalla __Submit to server__.

<img src="/static/images/installation/vscode/tmc_menu.png"
alt="TMC Menu"
width="100%" height="100%"/>

Kun tehtävä on lähetetty palvelimelle ja siihen liittyvät testit menevät läpi, saat tehtävästä pisteet sekä mahdollisen lyhyen kyselyn. Vastaathan aina kyselyihin, sillä vastaukset auttavat kurssien kehityksessä.

### Pistetilanteen tarkistaminen <a name="pistetilanteen-tarkistaminen"></a>

Voit tarkastaa oman pistetilanteesi kirjautumalla osoitteessa [tmc.mooc.fi](https://tmc.mooc.fi) olevaan verkkopalveluun. Verkkopalveluun kirjaudutaan Mooc.fi-tunnuksilla.
