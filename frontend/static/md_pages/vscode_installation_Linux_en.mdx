Follow the instructions below to download Visual Studio Code (VSCode) and TestMyCode (TMC) extension for VSCode. 

You need to have administrator priviledges on your computer to install the necessary programs.  
If you are using a computer provided by the computer science department you can skip directly to [Installing TestMyCode extension](#TestMyCode-asentaminen) section.

###  Installing Visual Studio Code <a name="VSCoden-asentaminen"></a>

You can install Visual Studio Code with the following terminal command (works for Ubuntu):

> `sudo apt-get install code`

If this doesn't work, please consult the Visual Studio Code installation instructions at [https://code.visualstudio.com/docs/setup/linux](https://code.visualstudio.com/docs/setup/linux)

### Installing TestMyCode extension for VSCode <a name="TestMyCode-asentaminen"></a>

1. Open Visual Studio Code by typing `code` in the terminal.
2. At the left you can see icons, choose the Extension view (see picture point 1).  
3. Type **TestMyCode** in the search field (see picture point 2).  
4. Press the search result **TestMyCode**.
5. Press "Install" (see picture point 3).  

The first initilization of the extension will download some necessary components (this will take a while).

### Installing Python <a name="python-asentaminen"></a>

You will need Python3 to run your source code. Python3 pip you will need later on the course, so please install it as well.

You can install Python3 and Python3 pip with the following terminal command (works for Ubuntu):

> `sudo apt-get install python3 python3-pip`

### Installing Java environment <a name="javan-asentaminen"></a>

You will need Java Development kit & Runtime Environment and Maven to run your source code for courses using Java.  
Please install Java following the instructions below.

#### Installing Java

##### 1. Install OpenJDK-package.

You can install OpenJDK with the following terminal command (works for Ubuntu):

> `sudo apt-get install openjdk-11-jdk`

##### 2. Switch the active version of Java

Use the command:

> `sudo update-alternatives --config java`

Type the number of the option, which says Java 11 (for example on this computer "/usr/lib/jvm/java-11-openjdk-amd64/bin/java"), and press enter.

If there are multiple options with Java 11, both with status "auto mode" and "manual mode", select "manual mode".

#### Installing Maven

You can install Maven by following the instructions below.  
**NB!** In commands, change the version number to match the downloaded version.

1. Download the latest Source tar.gz archive, `apache-maven-3.6.3-src.tar.gz`, from [https://maven.apache.org/download.cgi](https://maven.apache.org/download.cgi).
2. Extract the archive to your desired location, e.g. `/usr/local/`, this will create automatically a folder named `apache-maven-3.x.y`.
3. Open terminal and run the following command, remember to change PATH to correct location:  
`export PATH=/usr/local/apache-maven-3.x.y/bin:$PATH`
4. Make sure that Maven is installed correctly by running the command `mvn --version`
