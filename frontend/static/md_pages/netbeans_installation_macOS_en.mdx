### 1.Installing Java <a name="javan-asentaminen"></a>

1. Download the open development environment for Java 11, AdoptOpenJDK11, from [https://adoptopenjdk.net](https://adoptopenjdk.net/?variant=openjdk11&jvmVariant=hotspot).

2. Select OpenJDK 11 (LTS) and HotSpot. Then click the "Latest release" button to download Java. 

3. When the download is complete, open the installation file. 

4. Follow the installation instructions to install Java.

If in phase 3 you get a notification saying **"OpenJDK can't be opened because it is from an unidentified developer."**, 
navigate to the directory you downloaded the installation file to in Finder, right-click the filename and select "Open" (see images below).

###  2. Installing TMCBeans <a name="tmcbeansin-asentaminen"></a>

For this step, you must have macOS 10.13 or newer. If you have an older version of macOS, download the zip version of TMCBeans by following the instructions in the ZIP tab.  

1. Download file [https://download.mooc.fi/tmcbeans/installers/tmcbeans_1.3.0_installer.dmg](https://download.mooc.fi/tmcbeans/installers/tmcbeans_1.3.0_installer.dmg).
2. When the download is complete, open the file. The following window should open: 

<img src="/static/images/installation/macos-tmcbeans-installer.png"
alt="Näkymä tmcbeans.dmg asennusohjelmasta, joka on auki Finderissä"
width="100%" height="100%"/>

Jos näkymä ei aukea, saat sen esiin painamalla Finderin sivupalkista sinne ilmestynyttä tmcbeans-levyä (katso allaoleva kuva).

<img src="/static/images/installation/macos-tmcbeans-finder-sidebar-dmg.png"
alt="Finderin näkymä, jossa on tmcbeans-levy mountattuna vasemmassa sivupalkissa."
width="100%" height="100%"/>

3. Vedä aukeavassa näkymässä tiedosto tmcbeans.app kansioon Applications.
4. Voit nyt sulkea dmg-asennustiedoston Finderin vasemmasta reunasta painamalla eject-nappia (katso allaoleva kuva).

<img src="/static/images/installation/macos-tmcbeans-finder-sidebar-eject-dmg.png"
alt="Finderin näkymä, jossa on ympyröitynä tmcbeans-levyn viereinen eject-nappi Finderin sivupalkissa."
width="100%" height="100%" />

Jos saat ilmoituksen **"tmcbeans.app can't be opened because it is from an unidentified developer"** kun yrität avata TMCBeansin, mene Finderissä Applications-kansioon ja koita avata TMCBeans painamalla sitä hiiren oikealla näppäimellä ja valitsemalla Avaa/Open. (katso allaolevat kuvat)

<img src="/static/images/installation/macos-tmcbeans-not-trusted.png"
alt="MacOS:n ilmoitus siitä, että tmcbeans.app-tiedostoa ei voitu avata koska se ei ole luotetulta lähteeltä."
width="100%" height="100%" />

<img src="/static/images/installation/macos-tmcbeans-rightclick-open.png"
alt="Finderin näkymä Applications-kansiosta, jossa tmcbeans.app-tiedostoa on painettu hiiren oikealla näppäimellä ja pudotusvalikosta on korostettuna vaihtoehto Open."
width="100%" height="100%" />
